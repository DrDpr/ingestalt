# Ingestalt User Guide

Welcome to the **Ingestalt** documentation. Ingestalt is a local-first, spatial OS designed for "Editorial Engineering"—a workflow that treats software architecture as a living document.

## 🚀 Getting Started
1. **Import your project**: Connect your local folder and run Discovery to auto-populate nodes from Markdown.
2. **Organize spatially**: Re-order nodes, group logical systems, and draw connection edges directly on the grid.
3. **Curated Curation**: Hydrate and enrich your nodes with rich attributes (Schemas, Interfaces, checklists).
4. **Task Engineering**: Use the AI Task Prompt Builder to feed surgical, up-to-date context into your IDE's AI assistant.
